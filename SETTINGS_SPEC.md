# Recorpaster 设置面板 Spec（Phase 2 · v1）

## 范围
菜单「设置…」打开一个设置窗口，含下列设置 + JSON 持久化 + 尽量实时生效（不重启）。
**不做（later）**：模型切换 / 下载管理、真 VAD 分段流式。

## 窗口
- SwiftUI 设置窗口，从菜单「设置…」打开。原生 macOS 设置风格，单页滚动 + 分区即可（暂不用多 tab）。标题「Recorpaster 设置」。可调大小。
- ⚠️ 本 app 是 LSUIElement（accessory，无 Dock 图标）。打开设置窗要处理激活：临时 `setActivationPolicy(.regular)` + `NSApp.activate`，让窗口能前置/获得焦点；关闭时切回 `.accessory`。否则窗口可能不前置或无法聚焦（menu-bar app 经典坑）。
- 注意：**不抢焦点的红线只针对浮条 HUD**；设置窗口是正常窗口，允许获得焦点。

## 持久化（扩展 Config.swift）
- 单一 JSON：`~/Library/Application Support/Recorpaster/config.json`。
- 启动时读；**缺失/损坏 → 用默认值，绝不崩**。
- 改动即存。
- 各项尽量热生效，不重启：
  - 快捷键 → 实时重注册 CGEvent tap（新 keyCode）。
  - 输出方式 → 粘贴时读。
  - 语言 → 下次 transcribe 传入 WhisperKit。
  - 标点开关 → 后处理时读。
  - 强调色 → 实时传给浮条 view。
  - 开机自启 → 立即 SMAppService 注册/注销。
- **改设置不得打断正在进行的听写**；快捷键重注册要在安全时机（非捕获中、非听写中）。

## 设置项（v1）

### 1. 快捷键（hold-to-talk）
- 默认右⌥（keyCode 61）。
- 「录制快捷键」按钮 → 捕获下一次按键（提示「按下要用的键…」），存 keyCode。
- v1 **限定修饰键类**（左右⌥ / 左右⌘ / fn / Control），避免吃掉正常打字；普通字母/空格不允许。
- 捕获期间暂停 tap，别让录键本身触发听写。
- 改完实时重注册 HotKeyMonitor，不重启。

### 2. 语言
- 选项：自动检测 / 中文 / 英文。默认**自动**。
- 传入 WhisperKit DecodingOptions.language；强制时更快更准。
- 与标点联动：BERT 标点仅中文；语言=英文时**跳过**标点后处理（英文 Whisper 自带标点）。沿用「仅对含汉字输出做 CJK 标点」的现有逻辑。

### 3. 标点
- 开/关，默认**开**。
- 关 = 跳过 BERT 后处理，输出原始；开 = 现行为。

### 4. 输出方式
- 选项：自动粘贴（⌘V） / 仅复制。默认**自动粘贴**。
- 自动粘贴 = 现行（剪贴板 + ⌘V + 还原），需辅助功能（已授）。
- 仅复制 = 只放剪贴板不粘，用户手动 ⌘V，无需辅助功能。
- （可选 later）模拟键入：CGEvent 逐字键入。⚠️ **中文经合成按键不稳——这正是当初改用剪贴板的原因**。若做，UI 标「适合英文，中文可能不稳」。v1 可不做。

### 5. 强调色
- 浮条 accent（Step 1 留的可覆盖属性挪来这调）。几个预设 + 自定义（NSColorPanel）。默认现行蓝。实时生效到 FloatingView。存 hex/RGBA。

### 6. 开机自启
- 开关，默认**关**。`SMAppService.mainApp` 注册/注销，立即生效。

### 7.（只读）当前模型 + 加载进度
- 显示当前模型名 + 加载进度（复用进度条那条工作的状态）。v1 **不切换**。

## 可选小件（便宜就做，低优先）
- **提示音**：开始/停止听写时轻提示音。开关，默认关。
- **实时逐字预览开关**（现 `RECOR_NO_STREAM` env → UI 开关）。默认开。

## 不变约束
- 浮条仍不抢焦点（红线只管 HUD）；设置窗口可聚焦。
- 别破坏现有闭环（转写 / 标点 / 粘贴 / 尾句）。
- 配置损坏 → 默认值，不崩。
- 所有 UI 文案中文。
